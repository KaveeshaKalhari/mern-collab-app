# MERN Collab App

A collaborative rich-text document web app built with MongoDB, Express, React, Node.js and Tailwind CSS.

## Tech Stack
- MongoDB Atlas, Express.js, React.js, Node.js, Tailwind CSS

## Setup

### Backend
```bash
cd server
npm install
cp .env.example .env
npm run dev
```

### Frontend
```bash
cd client
npm install
npm start
```

## Environment Variables
See `server/.env`
